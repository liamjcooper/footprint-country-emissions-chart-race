## Install backend

Clone this repo, then

```bash
npm i
```

## Run backend

The backend runs on port 8000

```bash
npm run backend:start
```

## Run backend tests

To run test the backend needs to be running.

```bash
npm run backend:test
```

## Run frontend

The frontend runs on port 5173

```bash
npm run frontend:start
```
