# Emissions per country per year chart race

## Install backend
Clone this repo, then
```bash
cd backend && npm i
```

## Run backend
The backend runs on port 5000, go to `http://localhost:5000/` to 

```bash
npm start
```

## Run backend tests
To run test the backend needs to be running.
```
npm test
```

## Install frontend
```bash
cd frontend && yarn
```

## Run frontend
```bash
yarn dev
```
